## Install

Clone the repo and install dependencies:

```bash
git clone --depth 1 --branch main https://github.com/ahmedmukhtar1133/exercise-kaleidoscope.git exercise-kaleidoscope
cd exercise-kaleidoscope
npm install
```

## Starting Development

Start the app in the `dev` environment:

```bash
npm start
```

## Packaging for Production

To package apps for the local platform:

```bash
npm run package
```
